== Description ==

WeCPi CE allows you to import products from CSV file. You have your products in a spreadsheet or database and want to be able to export a CSV file and import your products automatically into WP e-Commerce. WeCPi is for you.

WeCPi CE does not include image import.

== Installation ==

1. Download and unzip to your plugin folder.
2. Activate the plugin from the Plugins section of your dashboard.
3. Test the plugin using the example data provided including a CSV file. Read the README in the plugins example_data directory.
